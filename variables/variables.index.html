Variables

What are variables?

Variable are a way of giving a name to a value and store it in order to reuse it. We can think of it as a jar with a label in. How do we do this? The basic syntax is let year = 1985. Here we are saying to JavaScript store the number 1985 to the variable year. This is called declaring a variable! We use the keyword let, variable name and then assign the value.

Updating Variables

How do we update variables? There is a shorthand syntax we can use to update a variable. Here is an example using score: score += 7. It is also super common to see the increment and decrement operator which is basically ++ and --.

I++ vs ++I

A common source fo confusion for beginners is often the difference between i++ and ++i. When we use i++ the code will be executed, but the value returned will be the value prior to expression. The prefix operator increments the value first and then returns the new updated value.

Const

Const means constant! This is value that cannot be changed. If we try to change a const value we will get an error message! For example pi will always be 3.14 and days of the week will always be 7. When we get to arrays and objects const is super useful!

Var

This is an old way to declare variables. It works in a similar way to let. This was orginally the only way to declare varaiables. You should not use it when writing modern JavaScript.

Booleans

Booleans are used to store yes or no values. They pay an important role in using logic in our programming and are stores as true or false. Variables can store booleans. Variables can also change type so a boolean could become a number.

Naming Conventions

When naming a variable, creating an identifier there are some hard rules and best practice to follow...

  • Must not begin with a number
  • Variable names should be in camelCase!
  • First leter is not capital
  • Do not give obscure names like n or x.
  • Create meaningful names
  • Boolean names generally start with is e.g isGamerOver
