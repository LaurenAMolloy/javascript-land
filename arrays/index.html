Arrays

Data Structures in JavaScript

A data structure is a structure containign a collection of data. For example we can have a variable that holds four different strings in a single array or a collection of ordered data. Examples of arrays include playlists, levels in a game, and comments on posts. How do we make an array? We use square brackets on the right side of an equal sign. Do not try and check the type of array using typeOf! You will have a bad time.

Arrays

Const is commonly used to initiate an array. If we take a look in dev tools. We can see the contents of our array. An array can hold multiple data types. Arrays have a built in property called length. Similar to a string.

How do we access elements in an array?

Let's talk about getting data out of an array! Arrays are zero indexed, each element has it's own index and we start counting at zero. These indices are important beacuse we can access elements using this index. The length is always one more than the index. In the example we can see length is 3, but if we try to access an element and index 3 we get undefined! We can update an array like so...days[3]="thursday"

Adding and Removing Elements

We can use the following built in array methods to add or remove elements to an array..Think of the array as a stack, we add one to the end using push and use pop to remove one. Shift is to remove one for the start and unshift adds an element. Shift returns the element that is removed.

  • push-add to the end
  • pop-remove from the end
  • shiftremove from the start
  • unshiftadd to the start

Examples

concat, indexOf, Includes, Reverse

Concat

Concat can be used to join two arrays together

Includes

Includes will return a boolean value and can be used to see if an array includes a particularc element.

indexOf

indexOf will return -1 if the element is not in the array and 1 if it is found.

Reverse is a destructive method and will return a reversed array

Slice

Slice will return a portion of an array. We can pass in a starting point and then we can get a portion from the element at the starting index to the last index. If we pass in a second argument, an end, it will stop but not include that index.

Splice

When we use splice it is a descrutctive methods. We pass in three arguments(start, amount of elements to delete, element to insert). The orginal is mutated and we will get the new array returned.

Sort

How can we sort an array? We need a compare function. The default sort method will convert everything to strings and compare the UTF-16 code unit values. This creates unexpected results.

Our Friends the Equality Operators

If we compare an array with another array using the strictly equal operator we will get false. Why is this? Well this is because JavaScript is not comparing the values inside. What JavaScript cares about is the reference in memory. If we store a number we will use a certain amount of memory. But with arrays this is not the case. Array are stored differently. Arrays are stored using an address that points to the contents. [1, 2, 3] === [1, 2, 3]//false.

Const and Arrays

When we make an array and use const we can update the content of the array. As long as the reference remains we can change the contents without getting an error. We cannot reassign the reference.

Multidimenional Arrays

Nested arrays are great for games like tic tac toe. We access the elements in exactly the same way. gameBoard[1][1]. We just chain square brackets to access the elements.
