What is JavaScript?

What can JavaScript be used for?

JavaScript is used to make website dynamic, so the user can interact with the webpage. We can make things show, hide, move, when the user scrolls a change happens. For example we can access current data about teh stock market and the page will update in realtime. We can choose a movie, watch a movie, like a movie on Netflix. We can go shopping! All of this is thanks to JavaScript ❤️.

Primitive Types

Types are a concept of any programming language. Types are used to store information. We use types to represent data such as textual values, numbers, yes or no (true or false) or empty values. These are known as primitive types and are the building blocks of JavaScript!

  • Number
  • String
  • Boolean
  • Null
  • Undefined

Technically symbol and BigIntare also primitive types!

Numbers

JavaScript has one number type. Other languages you have two types usually integers and decimals(floats). Numbers have a hard limit in terms of space in memory. We can perform math operations using JavaScript such as...

  • addition +
  • subtraction -
  • multiplication *
  • division /
  • module %

JavaScript follows the PEMDAS order of operation. Parenthesis, Exponents, Multiplication, Division and Addition.

Module

Modulo is known as the remainder operator so for example 19 % 4 will give us 3.

Exponent

This 2 ** 4 is basically 2 to the power of 4.

NaN

NaN is a number, but represent a value that is not a number. So for example if you use typeOf NaN this will return a number.
