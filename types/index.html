What is JavaScript?

What can JavaScript be used for?

JavaScript is used to make website dynamic, so the user can interact with the webpage. We can make things show, hide, move, when the user scrolls a change happens. For example we can access current data about the stock market and the page will update in realtime. We can choose a movie, watch a movie, like a movie on Netflix. We can go shopping! All of this is thanks to JavaScript ❤️.

Primitive Types

Types are a concept of any programming language. Types are used to store information. We use types to represent data such as textual values, numbers, yes or no (true or false) or empty values. These are known as primitive types and are the building blocks of JavaScript!

  • Number
  • String
  • Boolean
  • Null
  • Undefined

Technically symbol and BigInt are also primitive types!

Numbers

JavaScript has one number type. Other languages you have two types usually integers and decimals(floats). Numbers have a hard limit in terms of space in memory. We can perform math operations using JavaScript such as...

  • addition +
  • subtraction -
  • multiplication *
  • division /
  • module %

PEDMAS & MODULO

JavaScript follows the PEMDAS order of operation. Parenthesis, Exponents, Multiplication, Division and Addition. Modulo is known as the remainder operator so for example 19 % 4 will give us 3.

NaN

NaN is a number, but represent a value that is not a number. So for example if you use typeOf NaN this will return a number.

Null and undefined

There are two primitive types which are very simple compared to strings. Both are single values like true or false. Undefined is returned when we try to access a value that is not there. For example const userName. Undefined is something we will run into very often! Whereas null is the intentional lack of a value. For example if a user ins not logged in, but we expect one at a later date we can do this...let loggedInUser = null

The Math Object

The math object contains properties for math constants and functions! To access these we can use the following syntax Math.floor(). Math.floor will take a number and chop off the decimal. This is not rounding for this we need Math.round(). We are lookign at Math.random() and this gives us a random number between 0 and 1. Most of the time we want a whole number, not just a decimal. Let's say we want a random number bewteeen 1 and 10. This is the approach we will take..Math.floor(Math.random() * 5) + 1. We add 1 because Math.random starts at 0!
