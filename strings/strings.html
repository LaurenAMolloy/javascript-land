Strings

What are strings?

A string is a string of characters, a primitive type wrapped in quotes. Strings are used to store textual information. In a single file choose whether to use double or single quotes to wrap your strings. Be consistent!

string.js 
let userName = "Lauren"

Indices and Length

Strings are indexed. This means that each letter has a corresponding index (a positional number). Why does this matter? This matters because we can use this positioning to access individual characters. We can do this using square brackets. Every string has a built in propery called .length. We can use this to find the number of characters, not the same as the index. The index is always one less because it starts with 0. We can also add strings together, this is called concatenation.

String methods

There are many methods that we can use on strings. We can change it to lowercase, uppercase and remove elements. This is how we implement it. thing.methd(). Strings are immutable so we are making a new version with these methods. Don't forget the parens at the end to make sure you actually run the methods.

Naming Conventions

When naming a variable, creating an identifier there are some hard rules and best practice to follow...

  • Must not begin with a number
  • Variable names should be in camelCase!
  • First leter is not capital
  • Do not give obscure names like n or x.
  • Create meaningful names
  • Boolean names generally start with is e.g isGamerOver
