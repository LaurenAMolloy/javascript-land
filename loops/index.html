Getting Loopy!

Do I need to repeat myself?

The whole point of a loop is to repeat some sort of functionality. A real world example of a loop would be comments to be displayed under a post. All the comments in the array will be looped over and displayed on the page.

Our First For Loop

The syntax for a for loop looks like this: for(
[initial expression];
[condition];
[incrementExpression]
)

How does the for loop work?

First we create a variable called i, the middle part is a boolean and as long as it is true the loop will keep running, the final part is an increment operator that will update i.

Nested Loops

We can nested loops. In practice this means for every time the outer loop runs the inner loop will run for a full cycle. Our outer loop is running 10 time. We then have 4 interation for the inside loop. One common area when we use this loop is when we want to traverse a nested array.

While Loop

The while loop will run while a condition is true. Be warned it is super easy to make an infinite loop using this. For loops are the ptrefered methiod for a set number of items. A while loop is more useful for something that is variable. In the example code the condition will run until the user guesses the secret word.

The Lovely for of Loop

The for of loop is a lot better for iterating over arrays. The basic syntax for the For...of looks like this...for(variable of iterable){statement} Let's compare the for loop with the for of loop! Although the for...of is a cleaner syntax if you need the index you should probably choose a for loop. There are other iterables you can use the for...of loop with like strings!

Can we use for...of with Objects?

No way! An object is not actually considered an iterable. The for of loop just will not cut it. The for in is used for iterating over an object and will jsut give us the key. It is not that common these days.

There are other ways to get the values from an object such as using the built in methods such as Object.entries(objectName). This returns an array of key value pairs.
