THE DOM

What is the DOM? The DOM stands for the Document Object Model. At it's core we have a set of objects that represent all the elements on a page. It is our portal into creating interactivity on a page. All interactive elements on a page are powered by JavaScript behind the scenes.

The DOM

When our browser loads a web page, all the HTML elements are represented by the DOM. HTML elements go in...JS Objects come out. All of these objects have certain properties available. There is a relationship between these objects and how they are organised called a tree. This is a data structure. The top level object is called the document. The object has all of our properties and methods to help us manipulate the DOM! Use console.dir(document) to see it all!

Selecting DOM Elements

When we say select we mean to grab or retrieve and elemnt to do some specific task. For example we select a button so when a user clicks an action occurs. So first we select, then we manipulate! Here are some ways to acheive this...

  • getElementById
  • getElementByTagName
  • getElementByClassName

Get Elements by tagName

This will return all elements with a specific tag name so all img or all p. In the console we will see a HTML collection. It looks like an array but it is not. We can iterate over the collection. What is inside the collection? An element.

Elements by Class

We can add a class name to each element and use the class to retrieve all elements with the same class. If there are no elements with a specific class then we get an empty collection.

QuerySelector

Is an all in one method used to select a single element. It is swiss army knife selector. the syntax is similar. We select by using document.querySelector(".img"). This provides us with the first match. But if we want all the elements then we us document.querySelectorAll("#img"). When we use selectorAll we are given a NodeList. What is a NodeList? We can select all descendents just like when we are styling in CSS, this is because we use CSS selectors to pass into the querySelector method.
