Modern JavaScript

Default Params

Default params are used as default values if not arguments are passed in. For example in our rollDie function we can have a default number of side for our dice! Watch out because order matters! If you miss a value undefined will be returned.

Spread

Spread is like a swiss army knife. Allows an array to be expanded when passed into a function as arguments or array method.

Map

Map is very different, we can take an array and call the map method and then the return value from the function is stored in a NEW array that is constrcuted based on the original. This new array can then be stored in a variable.

Arrow Functions

Arrow functions are a newer compact version of writing a function expressions (a function stored in a variable). We donot require the function keyword and are super useful when we are using array methods and callback functions.

  • Remove the function keyword
  • Add an arrow to the right of the parameters!
  • If there are no arg leave empty parens()
  • If there is only 1 argument then we can elimate the parens completely
  • If there is 0 or 2 parameters then parens must be included!
  • Implicit return can be used by skipping {} for shorter one line functions
  • We can also replace the {} with () for implicit returns for multiline functions

Important! implicit returns only work when we have a single value to return.

setTimeout & setInterval

Settimeout allows us to execute a function at a later date! You pass two things, a classback and a delay is milliseconds. Another function that is similar to setTimeout is setInterval, this will continuously excute a function passed in at a specific interval. We can stop setTimeout using clearInterval. This uses an id coresponding to the interval we wish to stop!

Filter

We can use .filter() to create a new array with all elements that pass a test implemented by the provided function. It needs to be a boolean. Any element that returns true will be added to the new array! We are never changing the original array. We can also chain methods such as map() to get specific results such as movie titls of high rated movie like so...movies.filter(m => m.score > 80).map(m => m.title);

Some and Every

We can use some to return true if ANY elements passes a test and every if EVERY element passes!

The Dreaded Reducer

The main goal of reduce is to take an array an reduce the elements to a single value. We need to be the ones to decide how to do this! Lets walk through this..

  • We provide a reducer function
  • The first parameter represents the accumulator
  • The second paremeter represents the current value
  • We return the value that we be used as the current accumulator on the next iteration
